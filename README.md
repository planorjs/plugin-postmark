# planor-plugin-postmark
Planor Postmark plugin.

## Usage
```js
import PlanorServicePostmark from '@planorjs/plugin-postmark'

const credentials = {
  token: '...',
  from: 'test@test.com'
}

const service = new PlanorServicePostmark(credentials)
```
